#Installation
1. Clone the repo:
   *git clone https://github.com/mbsurihd/TeleDown*
3. *cd TeleDown*
4. Run install.ps1:
   *./install.ps1*
5. Setup config.json using schema
[You might have to change execution policy. Googleit if you need to.]

#Usage:
Run run.ps1
  *./run.ps1*
OR
Activate venv and run *python main.py*
